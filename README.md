# Simple Portfolio Website

This is a basic portfolio website for testing purposes.

## How to View

Open `index.html` in your web browser.

Alternatively, to run a local server (recommended for testing):

1. Open a terminal in this folder.
2. Run `python3 -m http.server 8000`
3. Open http://localhost:8000 in your browser.

## Customization

Edit `index.html` to change content, and `styles.css` for styling.
